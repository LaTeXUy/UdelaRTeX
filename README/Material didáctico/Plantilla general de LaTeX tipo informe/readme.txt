====  Algunas breves recomendaciones sobre latex   ======

Un buen manual: http://tug.ctan.org/tex-archive/info/lshort/english/lshort.pdf
y su version en español (no tan completa): http://tug.ctan.org/tex-archive/info/lshort/spanish/lshort-a4.pdf

Algunos programas necesarios:

 --- EN WINDOWS
Copilador lateX:   MikTex  (+ paquetes necesarios que se bajaran automaticamente)


 ---EN LINUX
instalar paquetes:
Latex:          texlive (mas paquetes con terminaciones similares: texlive-extra, texlive-fonts, etc)

En ambos sistemas operativos se pueden usar cualquiera de estos editores.
Editor latex:     Texstudio o  TexMaker


- lista de archivos incluidos:
 plantilla_latex.tex
 plantilla_latex.pdf
 ./figuras/mate.jpg
 ./figuras/surf.pdf
 README.txt

Abrir el archivo .tex con el editor de latex, por ejemplo texstudio y ejecute el comando PdfLaTeX y luego busque el pdf generado en la misma carpeta del tex.
